CARBON FUNCTIONAL MATERIALS DATABASE (cfms_db)
-----------------------------------------------------------------------------------------
VERSION:
The cfms_db.csv file is in version 0.1 and it was obtained from the processsing of 10,975
articles which contained the terms "biochar" or "hydrochar" either in the title, abstract
or keywords. The title of each article indexed in the cfms_db.csv is detailed in the file
filename_id_1.1.csv.
-----------------------------------------------------------------------------------------
The carbon functional materials database (cfms_db) was generated from the processing of 
scientific articles by the a.RIX engine (https://github.com/amaurijp/aRIX). Details on 
the use of the a.RIX engine is presented in:

"An automated domain-independent text reading, interpreting and extracting approach for
reviewing the scientific literature"

Amauri J. Paula

Solid-Biological Interface Group (SolBIN), Department of Physics, Universidade Federal
do Ceará – UFC, P.O. Box 6030, Fortaleza, CE, 60455-900, Brazil

https://arxiv.org/abs/2107.14638

The features (columns in the cfms_db.csv file) extracted are:

1. carbonization (synthesis) method / Physical Unit = None (categorical terms has no PU); 
2. biomass precursor / Physical Unit = None (categorical terms has no PU); 
3. carbonization temperature / Physical Unit = temperature; 
4. carbonization time / Physical Unit = time; 
5. surface area / PU = area weight-1; 
6. particle size / Physical Unit = Euclidean distance; 
7. adsorption capacity / Physical Unit = weight weight-1;
8. H/C weight ratios / Physical Unit = dimensionless;
9. C/N weight ratios / Physical Unit = dimensionless; 
10. O/C weight ratios / Physical Unit = dimensionless; 
11. C/O weight ratios / Physical Unit = dimensionless; 
12. N/C weight ratios / Physical Unit = dimensionless; 
13. C/H weight ratios / Physical Unit = dimensionless;
14. high heating value (HHV) / Physical Unit = energy weight-1;
